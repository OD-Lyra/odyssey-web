# Conventions

## Strict English Rule

For global scalability:

- All **variables**, **functions**, **classes**, **filenames**, and **module names** must be **English**.
- All **comments** must be **English**.
- UI copy can be localized (FR/EN), but implementation stays English.

## Brain vs Engine

- `app/brain/`: Intelligence (IP). Decision-making and AI logic.
- `app/engine/`: Execution. Deterministic media processing and rendering.
